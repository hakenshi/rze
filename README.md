# rze

`rze` is a wallpaper-driven colorscheme generator and template renderer.

This repository is intentionally documented primarily for implementers (see `docs/`).

- Start here: `docs/vision.md`
- Architecture overview: `docs/architecture.md`
- Pipeline contract: `docs/pipeline.md`
- Template language + cookbook: `docs/templates.md`

Developer shortcuts (optional): install `just` and run `just`.
